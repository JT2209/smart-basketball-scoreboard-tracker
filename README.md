# smart-basketball-scoreboard-tracker

## Description

A web-based platform that allows users to manage scores, players, fouls, timeouts, 
and game timing for every basketball game in real time using a laptop or tablet.

## Project Summary

I came up with this project since I have an experience in playing basketball for Special Olympics
and WDSRA since 2014. Scorekeepers run basketball games where the clock, fouls, and timeouts are tracked 
instantly with a clean, profession digital display, with no expensive hardware necessary. With a simple interface, 
scorekeepers can update scores, control the game clock, and track key stats while spectators view a  live scoreboard on 
a large screen. The system also gaves game history, making it simple to review past performances and improve team 
strategy. This project provides a low-cost, accessible alternative to traditional scoreboards while improving accuracy 
and organization. With future expansion potential such as live online viewing and a resposive design, this tool can grow 
into a powerful sports management platform. This is my first time doing this capstone project.
I took HTML and SQL classes at COD for eight years. This project will inspire me to recap what
the game recaps will look like after the finished display of the scoreboard.

## Features

- Game Setup
- Real Score Tracking
- Game Clock Control
- Foul and Timeout Tracking
- Live Scoreboard Display Mode
- Responsive Design
- Player Entry Area
- UI Design and Usability
- User Login
